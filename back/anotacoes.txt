- Primeiros passos:
1 . Instalar e configurar duas extensões para o vscode
    1.1 -> Prisma {
        fazer as configurações que estão na pagina inicial da extenção.
    }
    1.2 -> Primas - Insider

2 . Instalar dependencia do Prisma {
    
    2.1: yarn add prisma

    //Inicializar, para que seja criado 1 pasta prisma com "schema.prisma" e o arquivo ".env"
    2.2: yarn prisma init
}

3 . Configurar a DATABASE_URL da o arquivo ".env"
    3.1: Exemplo: DATABASE_URL="postgresql://sistema:gsap@manna@lgvhost.mannatech.com.br:3306/sistema?schema=public"

4. Configurar o arquivo "prisma/schema.prisma" dessa forma {
    generator client {
        provider = "prisma-client-js"
    }

    datasource db {
        provider = "mysql"
        url      = env("DATABASE_URL")
    }

}
